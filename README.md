# Witty Wolf Studio

Site oficial do Witty Wolf Studio - Design & Marketing Studio.

## Começar

Primeiro, instalar as dependências:

```bash
npm install
```

Depois, executar o servidor de desenvolvimento:

```bash
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000) no browser.

## Estrutura

- `/` - Página principal com Hero
- `/projetos` - Página de projetos
- `/contacto` - Página de contacto

## Tecnologias

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- React Icons
